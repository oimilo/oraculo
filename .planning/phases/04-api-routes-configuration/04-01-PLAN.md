    1. Create `src/app/api/stt/route.ts`:
       - Export async function `POST(request: NextRequest)`
       - Get API key via `requireEnv('OPENAI_API_KEY')` — catch and return 500 `{ error: "STT service not configured" }`
       - Parse request as FormData: `const formData = await request.formData()`
       - Get audio file: `const audioFile = formData.get('audio') as File | null`
       - Validate: if no audioFile, return 400 `{ error: "Missing required field: audio" }`
       - Build Whisper FormData: `const whisperForm = new FormData(); whisperForm.append('file', audioFile, audioFile.name || 'audio.webm'); whisperForm.append('model', 'whisper-1'); whisperForm.append('language', 'pt'); whisperForm.append('response_format', 'json');`
       - Call Whisper: `fetch('https://api.openai.com/v1/audio/transcriptions', { method: 'POST', headers: { 'Authorization': `Bearer ${apiKey}` }, body: whisperForm })`
       - If non-ok response, return 502 `{ error: `Whisper API error: ${status}` }`
       - Parse response JSON, return `NextResponse.json({ text: result.text })`
       - Wrap in try/catch, return 500 on unexpected errors

    2. Create `src/app/api/nlu/route.ts`:
       - Export async function `POST(request: NextRequest)`
       - Get API key via `requireEnv('ANTHROPIC_API_KEY')` — catch and return 500 `{ error: "NLU service not configured" }`
       - Parse JSON body expecting `{ transcript: string, questionContext: string, options: { A: string, B: string } }`
       - Validate: if transcript missing, return 400; if options missing or options.A/B missing, return 400
       - Build system prompt: `"You are a binary classifier for an interactive art installation called O Oraculo. Classify the visitor's response into one of two options. Return ONLY valid JSON with fields: choice (\"A\" or \"B\"), confidence (0.0 to 1.0), reasoning (brief explanation in Portuguese)."`
       - Build user message: `"Context: ${questionContext}\n\nOption A: ${options.A}\nOption B: ${options.B}\n\nVisitor said: \"${transcript}\"\n\nClassify this response as A or B. Return JSON only."`
       - Call Anthropic: `fetch('https://api.anthropic.com/v1/messages', { method: 'POST', headers: { 'Content-Type': 'application/json', 'x-api-key': apiKey, 'anthropic-version': '2023-06-01' }, body: JSON.stringify({ model: 'claude-3-5-haiku-20241022', max_tokens: 256, messages: [{ role: 'user', content: userMessage }], system: systemPrompt }) })`
       - If non-ok response, return 502 `{ error: `Anthropic API error: ${status}` }`
       - Parse response: `const result = await response.json(); const content = result.content[0].text;`
       - Parse the JSON from content text: `const classification = JSON.parse(content)`
       - Validate classification has choice (A or B), confidence (number), reasoning (string)
       - If parsing/validation fails, return fallback: `{ choice: 'A', confidence: 0.5, reasoning: 'Failed to parse classification response' }`
       - Return `NextResponse.json(classification)`
